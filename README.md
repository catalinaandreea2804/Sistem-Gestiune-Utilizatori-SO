# Sistem Gestiune Utilizatori (Bash)

Scripturi Linux pentru administrarea conturilor de utilizator.

## Ce face proiectul:

* **Inregistrare:** Creeaza conturi noi si valideaza datele (mail, telefon, parola).
* **Autentificare:** Sistem de Login si Logout securizat.
* **Rapoarte:** Calculeaza automat numarul de fisiere si spatiul ocupat de fiecare user.

---

## Cum se foloseste:

1. Ofera permisiuni de executie: 
   `chmod +x *.sh`
2. Porneste scriptul dorit: 
   `./inregistrare.sh` sau `./autentificare.sh`
